Jupyter installation and use guide

Write your first python notebook with Jupyter in a few minutes.

Author: Nikos Chasapis

Date: Mar. 28, 2021

What is a Jupyter Notebook?

Jupyter is an acronym meaning Julia, Python, and R. These programming languages were the original use case for Jupyter although it now supports many other languages. The main components to the whole environment are the notebooks themselves, the application, the notebook kernel and the notebook dashboard.

Jupyter is a server-client application, meaning it allows you to edit and run your notebooks via a web browser. The application can be executed on a PC without Internet access, or it can be installed on a remote server where you can access it through the internet.

A "notebook" is a document that contain both code and rich text elements, such as figures, links, equations, etc. Because of the mix of code and rich text elements, these files are the ideal place to bring together an analysis and its results. They can also be executed to perform the data analysis in real-time, meaning the code is split into blocks which can be run independently.

How to install Jupyter Notebook through Anaconda

One of the requirements here is Python 3.3 or greater, since Python 2.7 reached the end of its life on January 1st, 2020. The general recommendation is that you use the Anaconda distribution to install both Python and the Jupyter Notebook.

Anaconda's advantage is that you have access to over 720 packages that can easily be installed with Anaconda's package dependency and environment manager. You can follow the instructions for the installation of Anaconda by clicking here.

Further info on the installation and usage of Jupyter can be found here.

Running Jupyter and creating a new notebook

To run Jupyter you can either search for the application in your search bar or simply launch the Anaconda Navigator where all of conda's goods can be found in one place.

All you have to do from here is launch Jupyter, your default browser will create a new tab where you can find the application. You can open a notebook file from your PC and continue working on it or simply create a new one by clicking on the "new" button.

And finally, we have our very first Python3 Jupyter Notebook!
